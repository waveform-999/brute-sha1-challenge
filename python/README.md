# Usage

## Install python 3

```shell
brew install pyenv
pyenv install 3.8.0
```

## Run

```shell
python3 brute-sha1.py <sha1> <length>
```

### Example

```shell
python3 brute-sha1.py 5b11c3fa491d6b03e9742693ca34610c516031b8 4
```

### Output

```shell
Success! string = klmn, hash = 5b11c3fa491d6b03e9742693ca34610c516031b8
0.24553489685058594
```
